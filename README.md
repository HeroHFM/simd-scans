# SIMD Scans

SIMD algorithms for binary search.

## Build

```sh
$ ./build.sh
```

## Run

Type of test to run defined in macro in `main.cpp`.

```sh
$ ./simd > data.csv
```

## Plot

```sh
$ python3 plot.py data.csv [reference_algorithm_name]
```
